**Seekhna** is a premium web application designed to help intermediate speakers master verbal Hindi for social and professional connections. It focuses on bridging the language gap for Indian-origin individuals, enabling them to connect better with relatives and navigate social settings with confidence.

![Seekhna App](public/vite.svg)

## Features

### üìö Vocabulary Flashcards
- **Interactive Learning**: Browse curated categories like *Greetings*, *Family*, and *Professional Intro*.
- **Rich Content**: Cards feature Hindi script (Devanagari), transliteration (Hinglish), and English translations.
- **Text-to-Speech (TTS)**: Listen to the correct pronunciation of every phrase using the browser's native speech synthesis.
- **Pronunciation Practice**: Use the microphone to record your voice. The app uses Speech-to-Text (STT) to compare your pronunciation with the target phrase and provides instant feedback.

### üé≠ Conversation Scenarios
- **Roleplay Mode**: Engage in simulated, step-by-step conversations (e.g., "Meeting a Relative").
- **Interactive Dialogue**: The virtual partner speaks automatically. You respond by speaking the hint phrase to advance the conversation.
- **Real-world Context**: Scenarios are designed to mimic common social interactions.

### ‚ú® Premium Design
- **Aesthetics**: A deep Royal Blue and Saffron color palette inspired by Indian culture, set against a modern dark mode base.
- **Glassmorphism**: Translucent cards, smooth gradients, and modern UI elements.
- **Animations**: Fluid transitions and pulse effects for a polished user experience.

## Tech Stack
- **Framework**: React + Vite
- **Styling**: Vanilla CSS (Custom Properties & Design System)
- **APIs**: Web Speech API (SpeechSynthesis & SpeechRecognition)

## Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/avinayak14/Seekhna.git
    cd Seekhna
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Open in Browser**
    Navigate to your local web host to start learning!

## Browser Support
This application relies on the **Web Speech API**. For the best experience (especially for Speech Recognition), please use **Google Chrome**, **Microsoft Edge**, or **Safari**.
